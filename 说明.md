### 什么是 Athena++？

**Athena++** 是用于天体物理流体力学模拟的下一代计算框架。它并非简单的版本更新，而是对原有 C 语言版本 Athena 代码的 **完全重新设计与 C++ 重写**。它主要用于研究吸积盘、恒星形成、行星大气以及高能天体物理等前沿领域。

### 核心技术优势

  * **卓越的并行性能**：采用了基于“任务（Task-based）”的并行执行模型，打破了传统同步障碍，在数万个 CPU 核心上具有极佳的强扩展性。
  * **先进的网格技术**：
      * 支持 **自适应网格细化 (AMR)**，能够灵活处理跨尺度物理问题。
      * 支持动态负载均衡，确保大规模计算时的资源利用率。
  * **多物理场耦合**：
      * **MHD (磁流体力学)**：支持理想与非理想磁流体模拟。
      * **广义相对论 (GR)**：内置广义相对论流体与磁流体模块，适用于黑洞吸积等强引力场环境。
      * **辐射传输**：支持频率依赖或灰体辐射传输模拟。
  * **坐标系支持**：原生支持笛卡尔坐标、圆柱坐标、球坐标以及自定义曲面坐标系。

### 为什么选择 Athena++？

相比于其他代码，Athena++ 的 C++ 实现更加模块化，使得开发者可以更方便地添加自定义物理模块或初值条件，同时其底层对现代计算架构（如缓存优化）做了深度适配，计算效率极高。
